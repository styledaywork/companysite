# NexusForge — Company Website Walkthrough

## Overview

Built a complete **premium technology services startup website** using **Next.js 16**, **React**, **TailwindCSS 4**, and **Framer Motion**. The site features 9 pages, smooth animations, glassmorphism UI, and the mandatory brand color palette.

## Pages Built

| Page | Route | Key Features |
|------|-------|-------------|
| Landing | `/` | Animated hero, services grid, industries preview, tech strip, CTA |
| About | `/about` | Mission, animated stats, values grid |
| Services | `/services` | 6 detailed service cards with feature lists |
| Technologies | `/technologies` | Grouped tech stack with colored sections |
| Industries | `/industries` | 11 industry cards with HIPAA/Audit Ready badges |
| Security & QA | `/security` | Dark hero, 8 security + 6 QA service cards, trust indicators |
| Process | `/process` | 6-step animated timeline |
| Portfolio | `/portfolio` | 4 interactive project cards with metrics |
| Contact | `/contact` | Styled form, contact info, enterprise CTA |

## Design System

| Token | Value | Usage |
|-------|-------|-------|
| `brand-bg` | `#FDF8E2` | Page background |
| `brand-primary` | `#C4C3E3` | Primary sections, borders |
| `brand-accent` | `#FCDD9D` | Highlights, badges |
| `brand-dark` | `#1E1E2F` | Text color |
| `brand-secondary` | `#6C63FF` | CTAs, links, active states |
